# 天气

- 使用flutter 开发一款桌面版本的天气预报app

- 之前使用electron + react 开发了一款同样的 [传送门](https://github.com/tiandi0228/electron_weather)

- 使用的是和风天气的免费接口，key可以自己去申请

- 目前只有在macos环境下打包使用
